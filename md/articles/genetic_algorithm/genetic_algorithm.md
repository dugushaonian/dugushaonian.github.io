# 一、进化论

达尔文在1859年出版的《物种起源》一书中详细的阐述了他的进化学说。主要论证了两个问题：

1. 物种是可变得，生物是进化的。
2. 自然选择是生物进化的动力。

这都不是本文的重点，在遗传算法中，我们需要理解的生物学只是主要有以下几点：

* 基因：

物竞天择，适者生存。
